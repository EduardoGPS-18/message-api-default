## Dependencies

This app is configured on so its installation is recommended [Docker](https://www.docker.com/)<br>
And you need to configure .env file, based on .env.example

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development (without docker)
$ npm run start:dev

# development (with docker)
$ npm run compose:dev

# production
$ command...

```
